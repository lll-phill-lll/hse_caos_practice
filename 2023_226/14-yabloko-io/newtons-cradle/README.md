# Колыбель Ньютона

## Инструкция по запуску

1. Склонировать репозиторий яблока: [gitlab.myltsev.ru/caos/hw/yabloko/](https://gitlab.myltsev.ru/caos/hw/yabloko/) и собрать его по инструкции.
2. Перейти в ветку с заданием по VGA: `git checkout task1-vga-symbols`.
3. Заменить содержимое файла solution.c на [solution.c](solution.c), также надо заменить [kernel.c](kernel.c).
4. `make`
